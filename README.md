# Mi primera página Web

Es mi primera creación página web tipo portafolio personal en mostrar mi presente perfil como estudiante de analisis programación. Desarrollado con HTML5, CSS3 y GitHub.

## Descripción

Muestra mi formación académica, habilidades técnicas, proyectos realizados y formas de contacto. Diseñado con enfoque responsive para que se vea bien en Computadores, tablets y celulares.

## Tecnologías Usadas

- HTML5; Estructura semántica
- CSS3: Flexbox y Grid 
- GitHub: Control de versiones.

## Secciones del Portafolio

1. Header - Navegación
2. Sobre Mí - Presentación
3. Habilidades - Tabla de tecnologías
4. Proyectos - Galería
5. Contacto - Formulario
6. Footer - Información

## Cómo Usar

1. Descarga la carpeta de mi proyecto
2. Abrir "index.html" en el navegador
3. Listo.
